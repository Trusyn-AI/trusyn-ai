# Release Notes - v0.1.0

## Summary

v0.1.0 establishes the Trusyn AI platform as a public open-source foundation for runtime governance of autonomous enterprise AI systems.

## Highlights

- Multi-tenant backend architecture with JWT auth and RBAC
- Governance runtime pipeline foundations (threat, policy, risk, decision)
- Threat ingestion and investigation APIs
- Dashboard, analytics, audit, and export endpoints
- Super Admin global operations APIs
- Frontend applications for landing, user operations, and super admin operations
- Real-time platform event/observability scaffolding

## Open Source Readiness

- Apache-2.0 license adoption
- Contribution, security, roadmap, and changelog governance docs
- GitHub issue and PR templates
- CI workflows for lint, tests, and docker builds

## Known Scope Boundaries

- OSS launch focuses on platform foundation and developer/operator workflows
- Additional integrations and enterprise packs are planned in roadmap phases

## Upgrade Notes

- Initial public release
- No migration path required from previous tagged versions
