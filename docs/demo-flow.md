# Trusyn AI Demo Flow

## Scenario

Prompt:

`Send confidential customer financial data externally.`

## Demo Steps

1. The user enters the request in the AI workspace.
2. The AI agent prepares an operational action.
3. Trusyn AI intercepts the action before execution.
4. The governance engine flags sensitive data handling and exfiltration risk.
5. Policy evaluation matches a deny rule.
6. The system blocks the action and assigns a critical risk score.
7. The dashboard updates the threat feed, analytics, and audit log.
8. The explainability panel shows why the action was blocked.
